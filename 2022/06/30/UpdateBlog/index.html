搭建部分

安装nodejs

从官网中直接安装的nodejs.
npm可能会遇到 npm WARN config global --global, --local are deprecated. Use `–location 报错

问题原因是 npm版本太低导致.

解决方案:
以管理员运行 PowerShell -> 输入 set-ExecutionPolicy RemoteSigned -> 选择 Y -> 输入npm-windows-upgrade -> 选择最新版本npm安装
